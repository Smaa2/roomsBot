What is RoomsBot?

RoomsBot creates a customizable category with a single voice channel in your server. Connect to the voice channel to create temporary rooms!

Multiple Modes

  • Random

    Random cool names.

  • Username

    The username of the person creating the room.

  • Prefix

    Custom name used as a prefix with random numbers.

Commands

Command Description User Standard Premium
add <@user>... Add user to room. Any
remove <@user>... Remove user from room. Any
reset Create a new RoomsBot category. Admin
prefix Change the bot prefix. Admin
naming ... Change rooms naming mode. Admin
